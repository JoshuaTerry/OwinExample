Simple Bearer Token Example

Run the project and open PostMan.

  1. Set your action to Post
  2. Set the url to: http://[VS Express Localhost URL]/api/v1/Login
  3. Under Headers add Key: Content-Type with the Value: application/x-www-form-urlencoded
  4. Under Authorization set Type to No Auth
  5. Under Body add the values in the picture below

When you send you should get back the response listed above. Copy the access_token returned here. Now in order to test the authorization do the following:

The user has the "Admin" and "General" role but does not have the "Exclusive" role.
If you call TestAdmin, you should see the message listed above "You are Admin".
If you call TestExclusive you will instead receive: "Message": "Authorization has been denied for this request."
